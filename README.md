# Favicon Snake

Experience the classic game known as Snake as a Favicon!

## Where to Play

[GitHub Pages](https://cpeyton78910.github.io/faviconSnake)

## How to Play

- Look at your browser tab icon (Above the URL Bar)
- Use WASD or Arrow Keys to control the snake
- Press Escape to pause and unpause the game
- Press Space to restart the game
- Eat apples to grow longer
- Be careful not to run into yourself; however, running into walls won't kill you
